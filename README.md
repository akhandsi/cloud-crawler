# Cloud Crawler
Crawl AWS or Azure webpages and create simple JSON inventory
# Installation

```
$ yarn install
```

or

```
$ npm install
```

# Running Project

```
$ yarn start
```

or

```
$ npm start
```

# Tools Used
- cheerio: Cherrio is used for web-crawling.
